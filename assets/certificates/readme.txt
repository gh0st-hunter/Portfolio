Certificates Folder
===================

This folder contains certificate images that will be displayed when users click on certification cards.

Add your certificate images with these filenames:
- cert1.jpg - Cybersecurity Foundation (Palo Alto Networks)
- cert2.jpg - Cryptography and Network Security (NPTEL)
- cert3.jpg - Android Bug Bounty Hunting (EC-Council)
- cert4.jpg - Cyber Defence Strategies (IT Masters)
- cert5.jpg - Practical Introduction to Cloud Computing (EC-Council)
- cert6.jpg - Introduction to Cybersecurity (Cisco)

Supported formats: .jpg, .jpeg, .png, .webp

Note: If you want to use different filenames, update the imagePath in /components/Certifications.tsx
